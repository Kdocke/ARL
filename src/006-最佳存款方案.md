# 最佳存款方案

**问题：**

​		假设一年整存零取的月息为 0.63% 。现在某人手里有一笔钱，他打算在今后的 5 年中的每年年底取出 1000 元，到第 5 年时刚好取完，请算出他存钱时应存入多少?

**分析：**

​		由后向前推算：则第 5 年年初存款数 = 1000/(1+12*0.0063)

```c
#include<stdio.h>

int main()
{
	int i;
	double money = 0.0;
	for(i=0;i<5;i++)
	{
		money = (money+1000.0)/(1+12*0.0063);
	}
	printf("应存入的钱数为:%0.2f\n",money);
} 
```
